本文档通过markdown语法规定的样式指定，基于诗歌经理做的markdown网页版渲染而成，适用的场景是：
- 用于在手机端写微信公众号。
- 其他任何支持富文本的，markdown样式渲染。

不过不完全支持所有markdown标记。现在也仅仅是够用而已。

## 标题
只有2级标题：
```
## // 一级标题，#、## 都是一级标题
### // 二级标题，### 及其之后的####、#####都是二级标题
```
## 粗体
*斜体*和**斜体**，修改了一下样式。
```
*斜体*
**粗体**
```

## 引用
```
> // 这个就是引用标签
```
> 就是这样的

### 超链接、有序列表、无序列表

[baidu](www.baidu.com)

展示如下：
[baidu](www.baidu.com)
